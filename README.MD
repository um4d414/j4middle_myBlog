# Учебный проект блога
## Функциональность:
- Создание поста
- Редактирование поста
- Удаление поста
- Поддержка изображений в постах
- Просмотр ленты постов
- Комментирование поста
- Редактирование комментария
- Удаление комментария

## Стэк:
- Java 21
- Spring Framework:
  - Web-MVC
  - JDBC
- H2 
- Thymeleaf
